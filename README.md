# 😄 Facial Emoji Representation (Emojify)

## 📌 Overview

This project detects human facial emotions in real-time using a webcam and displays a corresponding emoji based on the detected emotion.

It combines Computer Vision and Deep Learning to create an interactive and fun application.

---

## 🚀 Features

* Real-time face detection using webcam
* Emotion recognition (e.g., happy, sad, angry, surprise)
* Emoji overlay based on detected emotion
* Simple and interactive interface

---

## 🛠️ Tech Stack

* Python
* OpenCV
* Deep Learning (CNN)
* NumPy

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/achieveramrit/facial_emoji_representation.git
cd facial_emoji_representation
```

## ▶️ Usage

Run the application:

```bash
python gui.py
```

* Your webcam will open
* The model will detect your face
* Emotion will be predicted
* Corresponding emoji will be displayed

---

## 🧠 How It Works

1. Face is detected using OpenCV
2. Detected face is preprocessed
3. CNN model predicts emotion
4. Matching emoji is displayed on screen
